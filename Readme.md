# Supply Chain Analytics Agent

An AI-powered supply chainanalytics agent that processes natural language questions and generates powerful analysis and SQL queries to analyze data using DuckDB, with a Streamlit-based user interface.

---

## Prerequisites

Make sure you have the following installed on your system:

- **Git**
- **Python (v3.11.9 only)**
  > Other Python versions are not supported.
- **pip** (Python package manager)

You can verify your Python version using:
python --version

## Local Setup Guide

# Step-1: Clone the repository

# Step-2: Create the .env file
Create a .env file in the root directory of the project and add:

        OPENAI_API_KEY=YOUR_OPENAI_API_KEY
        OPENAI_MODEL="gpt-4.1"

Replace:

        YOUR_OPENAI_API_KEY with your actual OpenAI API key
        gpt-4.1 with any supported OpenAI model if required

# Step-3: Creata a Virtual Environment
    python -m venv venv

# Step-4: Activate the Virtual Environment
    ./venv/Scripts/Activate.ps1
    
After activation, your terminal should show:

        (venv)

# Step-5: Install Required Dependencies
    python -m pip install -r requirements.txt

# Step-6: Run the Ingestion Pipeline
This step initializes and sets up the DuckDB database.

    python -m database.ingest

# Step-7: Launch the Streamlit UI
    streamlit run app/streamlit_app.py

Once started:
    Open the displayed local URL in your browser
    Enter your analytics question
    Wait for the agent to generate and execute the SQL query
    View the results in the UI

# Step-8: Manually Test Generated SQL (Optional)

If you want to manually test the SQL query generated by the agent:
    Copy the exact SQL query.
    Paste it inside database/runQuery.
    Stop the running Streamlit process (close all active terminals).

Run:
    python -m database.runQuery
